# Frame: WF-008 논문 학습 페이지 - 초기 상태

## 1. Visual Purpose

이 프레임은 사용자가 서재에서 선택한 논문을 중심으로 본문을 읽고, 같은 화면에서 AI 튜터와 질문을 주고받으며 학습을 시작할 수 있음을 보여준다.

## 2. Layout Regions

| Region ID | Region Name | Parent Region | Placement Notes | Region Purpose |
|---|---|---|---|---|
| R0 | Frame Root | None | 논문 학습 페이지 전체를 담는 화면 루트 | 글로벌 상단 바와 논문 학습 작업 영역을 함께 구성한다 |
| R1 | 글로벌 상단 바 | R0 | 화면 최상단에 가로로 배치 | 제품 로고, 상단 버튼, 사용자 프로필 진입점을 제공한다 |
| R2 | 학습 작업 영역 | R0 | 글로벌 상단 바 아래 전체 영역에 배치 | 논문 본문 읽기와 AI 튜터 대화를 한 화면에서 나눈다 |
| R3 | 학습 헤더 | R2 | 학습 작업 영역의 상단에 가로로 배치 | 서재로 돌아가는 버튼과 현재 논문 제목을 표시한다 |
| R4 | 접힌 목차 네비게이션 레일 | R2 | 학습 헤더 아래, 작업 영역의 가장 왼쪽에 좁게 배치 | 사용자가 목차 네비게이션을 열 수 있는 진입점을 제공한다 |
| R5 | 논문 원문 영역 | R2 | 접힌 목차 네비게이션 레일 오른쪽, AI 튜터 패널 왼쪽에 배치 | 선택한 논문의 현재 섹션 본문과 그림 등 실제 원문 내용을 보여준다 |
| R6 | AI 튜터 패널 | R2 | 학습 헤더 아래, 작업 영역의 오른쪽에 세로 패널로 배치 | 사용자가 논문 내용에 대해 질문하고 AI 답변을 확인하게 한다 |
