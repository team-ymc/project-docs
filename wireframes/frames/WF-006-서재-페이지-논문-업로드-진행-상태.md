# Frame: WF-006 서재 페이지 - 논문 업로드 진행 상태

## 1. Visual Purpose

이 프레임은 사용자가 서재에서 새 논문이 업로드 또는 분석 진행 중인 상태를 확인하고, 등록 과정이 아직 완료되지 않았음을 파악할 수 있음을 보여준다.

## 2. Layout Regions

| Region ID | Region Name | Parent Region | Placement Notes | Region Purpose |
|---|---|---|---|---|
| R0 | Frame Root | None | 웹 페이지 크기의 서재 화면 전체를 담는 루트 | 상단 글로벌 바, 서재 업로드 진행 상태, 하단 페이지네이션을 함께 구성한다 |
| R1 | 글로벌 상단 바 | R0 | 화면 최상단에 가로로 배치 | 제품 로고, 상단 버튼, 사용자 프로필 진입점을 제공한다 |
| R2 | 서재 상단 영역 | R0 | 글로벌 상단 바 아래에 배치 | 서재 제목과 논문 등록 버튼을 제공한다 |
| R3 | 서재 제어 영역 | R0 | 서재 상단 영역 아래에 가로로 배치 | 논문 검색과 보기 전환을 제공한다 |
| R4 | 업로드 진행 논문 영역 | R0 | 서재 제어 영역 아래에 단일 행으로 배치 | 업로드 후 분석 진행 중인 논문 1개를 보여준다 |
| R5 | 페이지네이션 영역 | R0 | 화면 하단 중앙에 배치 | 현재 서재 목록이 1페이지 상태임을 표시한다 |
