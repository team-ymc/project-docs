# Frame: WF-001 랜딩 페이지 - 비로그인

## 1. Visual Purpose

이 프레임은 비로그인 사용자가 Paper Teacher의 핵심 가치와 사용 시작 방법을 한눈에 이해하고, 로그인 또는 시작 행동으로 자연스럽게 진입할 수 있음을 보여준다.

## 2. Layout Regions

| Region ID | Region Name | Parent Region | Placement Notes | Region Purpose |
|---|---|---|---|---|
| R0 | Frame Root | None | 비로그인 랜딩 페이지 전체를 담는 영역 | 로고, 핵심 문구,로그인/회원가입 버튼을 한 화면 중앙에 배치한다 |

## 3. Component Inventory

| Component ID | Component Name | Region ID | Component Type | Component Purpose |
|---|---|---|---|---|
| C1 | 로고 아이콘 | R0 | Icon | Paper Teacher의 브랜드와 논문 학습 서비스를 시각적으로 보여줌 |
| C2 | 중앙 소개 문구 | R0 | Text | "논문을 이해하도록 가르치는 AI 리딩 튜터”라는 핵심 가치를 전달 |
| C3 | 로그인 / 회원가입 버튼 | R0 | Button | 비로그인 사용자가 인증 흐름으로 진입하게 한다 |
