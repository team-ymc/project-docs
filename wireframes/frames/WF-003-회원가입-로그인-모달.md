# Frame: WF-003 회원가입 / 로그인 모달

## 1. Visual Purpose

이 프레임은 비로그인 랜딩 페이지 위에 회원가입 / 로그인 모달이 오버레이로 열린 상태를 보여준다. 사용자는 배경의 랜딩 페이지 맥락을 유지한 채 인증 방법을 선택할 수 있다.

## 2. Layout Regions

| Region ID | Region Name | Parent Region | Placement Notes | Region Purpose |
|---|---|---|---|---|
| R0 | Frame Root | None | 비로그인 랜딩 페이지와 인증 모달 오버레이를 함께 담는 전체 화면 | 랜딩 페이지에서 인증 모달로 진입한 상태를 보여준다 |
| R1 | 비로그인 랜딩 페이지 배경 | R0 | 화면 전체에 배치되며 WF-001 랜딩 페이지가 뒤에 보임 | 모달이 어떤 화면 위에서 열렸는지 보여준다 |
| R2 | 오버레이 딤 영역 | R0 | 랜딩 페이지 배경 위에 반투명 레이어로 배치 | 사용자의 시선을 중앙 인증 모달로 모은다 |
| R3 | 인증 모달 | R0 | 화면 중앙에 오버레이로 배치 | 사용자가 로그인 또는 회원가입 방법을 선택하게 한다 |

## 3. Component Inventory

| Component ID | Component Name | Region ID | Component Type | Component Purpose |
|---|---|---|---|---|
| C1 | 랜딩 페이지 핵심 콘텐츠 | R1 | Background Page | WF-001의 로고, 중앙 소개 문구, 로그인 / 회원가입 버튼이 배경으로 유지됨을 보여준다 |
| C2 | 딤 처리 배경 | R2 | Overlay Backdrop | 배경 랜딩 페이지보다 인증 모달이 우선 상태임을 보여준다 |
| C3 | 닫기 버튼 | R3 | Icon Button | 모달을 닫고 비로그인 랜딩 페이지로 돌아가게 한다 |
| C4 | 모달 제목 | R3 | Text | 로그인 또는 회원가입을 진행하는 화면임을 알린다 |
| C5 | 안내 문구 | R3 | Text | "논문을 읽고 AI 튜터와 대화하려면 계속 진행해 주세요"라는 안내를 표시한다 |
| C6 | Google 계속하기 버튼 | R3 | Button | Google 계정으로 인증을 시작하게 한다 |
| C7 | Kakao 계속하기 버튼 | R3 | Button | Kakao 계정으로 인증을 시작하게 한다 |
| C8 | Naver 계속하기 버튼 | R3 | Button | Naver 계정으로 인증을 시작하게 한다 |
| C9 | 약관 동의 안내 문구 | R3 | Text | "계속하면 이용약관 및 개인정보 처리방침에 동의하는 것으로 간주됩니다"라는 안내를 표시한다 |
