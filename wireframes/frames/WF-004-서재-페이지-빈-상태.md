# Frame: WF-004 서재 페이지 - 빈 상태

## 1. Visual Purpose

이 프레임은 로그인한 사용자가 아직 등록한 논문이 없는 상태에서 서재가 비어 있음을 이해하고, 첫 논문을 추가해 학습을 시작할 수 있음을 보여준다.

## 2. Layout Regions

| Region ID | Region Name | Parent Region | Placement Notes | Region Purpose |
|---|---|---|---|---|
| R0 | Frame Root | None | 서재 페이지 전체를 담는 화면 루트 | 상단 글로벌 바와 빈 서재 콘텐츠를 함께 구성한다 |
| R1 | 글로벌 상단 바 | R0 | 화면 최상단에 가로로 배치 | 제품 로고, 상단 버튼, 사용자 프로필 진입점을 제공한다 |
| R2 | 서재 콘텐츠 영역 | R0 | 글로벌 상단 바 아래 전체 콘텐츠 영역 | 서재 제목과 빈 상태 안내, 논문 등록 행동을 보여준다 |
| R3 | 서재 콘텐츠 상단 바 | R2 | 서재 콘텐츠 영역의 상단에 별도 배경과 하단 구분선을 가진 가로 바 형태로 배치 | 서재 섹션의 제목과 콘텐츠 영역 상단 상태를 표시한다 |
| R4 | 첫 논문 등록 유도 영역 | R2 | 서재 콘텐츠 상단 바의 구분선 아래 본문 영역 중앙에 배치 | 등록된 논문이 없을 때의 안내와 논문 등록 행동을 보여준다 |
