# Frame: WF-005 논문 등록 - 모달

## 1. Visual Purpose

이 프레임은 서재 페이지 위에 논문 등록 모달이 오버레이로 열린 상태를 보여준다. 사용자는 서재 맥락을 유지한 채 PDF 파일 업로드 또는 URL 입력 중 하나를 선택하고 논문 등록을 시작할 수 있다.

## 2. Layout Regions

| Region ID | Region Name | Parent Region | Placement Notes | Region Purpose |
|---|---|---|---|---|
| R0 | Frame Root | None | WF-004 서재 페이지와 논문 등록 모달 오버레이를 함께 담는 전체 화면 | 서재에서 논문 등록 모달이 열린 상태를 보여준다 |
| R1 | 서재 페이지 배경 | R0 | 화면 전체에 배치되며 WF-004 서재 빈 상태가 뒤에 보임 | 모달이 어떤 화면 위에서 열렸는지 보여준다 |
| R2 | 오버레이 딤 영역 | R0 | 서재 페이지 배경 위에 반투명 레이어로 배치 | 사용자의 시선을 논문 등록 모달로 모은다 |
| R3 | 논문 등록 모달 | R0 | 화면 중앙에 작은 모달 형태로 배치 | 논문 등록에 필요한 상단, 입력, 업로드 실행 영역을 담는다 |
| R4 | 모달 상단 영역 | R3 | 모달 상단에 배치 | 제목, 닫기 버튼, PDF 업로드 / URL 입력 탭을 제공한다 |
| R5 | 등록 입력 영역 | R3 | 탭 아래에 점선 박스 형태로 배치 | 선택된 등록 방식에 따라 PDF 드롭존 또는 URL 입력 영역을 보여준다 |
| R6 | 등록 액션 영역 | R3 | 모달 하단에 전체 너비 버튼으로 배치 | 논문 업로드 또는 등록 실행 행동을 제공한다 |

## 3. Component Inventory

| Component ID | Component Name | Region ID | Component Type | Component Purpose |
|---|---|---|---|---|
| C1 | 서재 페이지 배경 콘텐츠 | R1 | Background Page | WF-004의 글로벌 상단 바, 빈 상태 안내, 논문 등록 버튼이 배경으로 유지됨을 보여준다 |
| C2 | 딤 처리 배경 | R2 | Overlay Backdrop | 배경 서재 페이지보다 논문 등록 모달이 우선 상태임을 보여준다 |
| C3 | 모달 제목 | R4 | Text | 현재 작업이 논문 등록임을 표시한다 |
| C4 | 닫기 버튼 | R4 | Icon Button | 논문 등록 모달을 닫고 서재 화면으로 돌아가게 한다 |
| C5 | PDF 업로드 탭 | R4 | Tab Button | PDF 파일 업로드 방식을 선택한 상태로 보여준다 |
| C6 | URL 입력 탭 | R4 | Tab Button | URL 입력 방식으로 전환할 수 있음을 보여준다 |
| C7 | 점선 업로드 박스 | R5 | Container | PDF 파일을 끌어다 놓거나 파일 선택 행동을 담는 입력 박스임을 보여준다 |
| C8 | 업로드 아이콘 | R5 | Icon | PDF 업로드 영역의 행동을 시각적으로 강조한다 |
| C9 | 드롭존 안내 문구 | R5 | Text | PDF를 끌어다 놓을 수 있음을 안내한다 |
| C10 | 파일 선택 버튼 | R5 | Button | 파일 선택 창을 통해 PDF를 고르게 한다 |
| C11 | 업로드 버튼 | R6 | Button | 선택된 PDF 업로드를 시작한다 |
