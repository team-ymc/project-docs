# Feature: FT-006 인라인 액션

## 1. Overview

- Summary: 사용자는 논문 뷰어에서 인라인 액션을 통해 번역 기능과 AI에게 질문하기 기능을 사용할 수 있다.
- Status: Draft
- Tracking: YMC-240
- Depends on: FT-004 논문 학습 뷰어

## 2. Feature Boundary

### In Scope

- 인라인 액션 창 열기: 본문 텍스트 선택 후 선택 영역 옆에 인라인 액션 창 표시 (WF-010)
- 번역하기: 선택한 텍스트의 인라인 번역 결과 표시 및 닫기 (WF-011)
- AI에게 질문하기: 선택한 텍스트를 맥락으로 AI 튜터 패널에서 질문 시작 (WF-TBD)

### Out of Scope

- AI 질문 작성·전송·응답 생성·대화 이어가기 → FT-007 AI 튜터 채팅
- 전체 본문 번역 → FT-005 전체 번역
- 인라인 노트 작성 → 후속(post-MVP)
- 번역 또는 AI 질문 진입 실패·재시도 → 후속(post-MVP)

## 3. Userflow References

| Userflow | Relation |

## 4. Stories

### Story 1. 사용자는 본문 텍스트를 선택해 인라인 액션을 열 수 있다 `MVP`

- type: USER
- Source Userflows: UF-005 Step 1-2
- Acceptance Criteria:
  - 사용자가 본문에서 텍스트 범위를 선택하면 선택 영역 옆에 인라인 액션 창(WF-010)이 표시된다.
  - 액션 창에는 번역과 AI 질문 진입 액션이 표시된다.
- Depends on: FT-004 Story 2

### Story 2. 사용자는 선택한 텍스트를 같은 자리에서 번역하고 본문 읽기로 돌아갈 수 있다 `MVP`

- type: USER
- Source Userflows: UF-005 Step 3-5
- Acceptance Criteria:
  - 사용자가 번역을 선택하면 선택 영역 옆에 인라인 번역 결과(WF-011)가 표시된다.
  - 사용자가 결과 창을 닫으면 원문 읽기 상태(WF-008)로 돌아간다.
- Depends on: Story 1

### Story 3. 시스템은 선택 텍스트와 문맥을 바탕으로 인라인 번역을 제공한다 `MVP`

- type: SYSTEM
- Source Userflows: UF-005 Step 3-4
- Acceptance Criteria:
  - 시스템은 선택 텍스트와 필요한 주변 문맥을 입력으로 번역 결과를 생성한다.
  - 생성된 번역 결과는 선택 영역 옆의 인라인 결과 창에 표시된다.
- Depends on: FT-003 Story 4

### Story 4. 사용자는 선택한 텍스트를 맥락으로 AI 질문을 시작할 수 있다 `MVP`

- type: USER
- Source Userflows: TBD
- Acceptance Criteria:
  - 사용자가 인라인 액션에서 AI 질문을 선택하면 AI 튜터 패널(WF-TBD)이 열리거나 활성화된다.
  - 선택한 텍스트는 AI 질문의 맥락으로 AI 튜터 패널에 전달되고, 사용자는 질문을 작성할 수 있다.
  - 질문 작성·전송과 응답 처리는 FT-007의 책임이다.
- Depends on: Story 1, FT-007 Story 1

## 5. Done Criteria

- 사용자는 본문 텍스트를 선택해 번역 또는 AI 질문 액션을 선택할 수 있다.
- 번역 결과를 인라인에서 확인하고 닫으면 원문 읽기로 돌아갈 수 있다.
- AI 질문을 선택하면 선택 텍스트가 AI 튜터의 질문 맥락으로 전달된다.

## 6. Open Questions

- AI 질문을 시작할 때 새 대화를 열지, 진행 중인 대화에 선택 텍스트를 추가할지, 사용자에게 선택지를 제공할지 정해야 한다.
