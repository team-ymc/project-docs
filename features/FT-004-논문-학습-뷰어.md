# Feature: FT-004 논문 학습 뷰어

## 1. Overview

- Summary: 분석된 논문 본문을 렌더링하고, 본문과 AI 패널이 함께 배치된 학습 페이지의 공통 레이아웃·상태를 제공하는 공통 캔버스다.
- Status: Draft
- Tracking: -
- Depends on: FT-002 서재, FT-003 논문 등록·분석

## 2. Feature Boundary

### In Scope

- 논문 학습 페이지 레이아웃 (본문 영역 + AI 패널 분할, WF-008)
- 구조화 본문 렌더링 (섹션 · 문단 · 그림 · 표 · 수식)
- 본문 스크롤 / 섹션 탐색
- 번역 · 인라인 · AI · 티칭이 얹히는 base 상태 관리

### Out of Scope

- 전체 번역 → FT-005 / 인라인 액션 → FT-006 / AI 튜터 채팅 → FT-007
- 위키 → FT-008 / 구조 맵 → FT-009 / 학습 기록 → FT-010
- 본문 편집 / 하이라이트 · 노트 저장 → 후속(post-MVP)

## 4. Stories

### Story 1. 사용자는 서재에서 선택한 논문을 본문·AI 패널 학습 페이지로 열 수 있다 `MVP`

- type: USER
- Source Userflows: UF-003 Step 1, UF-004 Step 1, UF-005 Step 1
- Acceptance Criteria:
  - `COMPLETED` 상태 논문 진입 시 본문 영역과 AI 패널이 함께 배치된 초기 상태(WF-008)가 표시된다.
  - FT-003 구조 추출 산출물(텍스트·그림·표·수식)이 섹션·문단 구조를 유지한 채 본문에 렌더링된다.
  - 본문은 번역/인라인/AI 기능이 참조할 수 있는 구조 단위를 유지한다.
